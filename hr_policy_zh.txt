HR政策手册 - 休假政策
I. 目的
本政策旨在明确公司为员工提供的休假福利，涵盖年假、病假和服务奖励假的相关规定。

II. 适用范围
本政策适用于公司所有正式全职员工，不分职位或部门。

III. 休假政策
A. 年假
1. 资格与累积：所有正式全职员工可享受年假。员工每月累积1.25天年假，每年最多15天。
2. 申请：休假申请须通过员工自助门户提前至少1天提交，需直属主管批准。
3. 未使用假期：未休年假可结转至下一年，但累计不得超过30天，超出部分自动作废。
4. 折现：未休年假可按基本工资折算现金（计算公式：基本工资÷30天×未休假天数）。
5. 试用期：试用期员工不享受年假。

B. 病假
1. 资格与累积：所有正式全职员工可享受病假。员工每月累积1.25天病假，每年最多15天。
2. 申请：计划性病假（如手术）需提前1天申请，突发疾病应尽快提交申请，均需直属主管批准。
3. 证明材料：连续病假超过2天需提供医疗证明。
4. 未使用假期：未休病假可结转至下一年，但累计不得超过30天。
5. 折现：病假不可折现。
6. 试用期：试用期员工病假减半（每月0.625天，每年7.5天）。

C. 服务奖励假
1. 资格与累积：服务满1年的正式全职员工可享受，每年5天。
2. 申请：提前1天通过系统提交，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：可按基本工资折算现金（计算方式同年假）。
5. 试用期：不适用。

D. 陪产假
1. 资格与累积：所有正式全职男性员工可享受，每次生育可休7天，最多享受4次。
2. 申请：尽量在预产期前1周提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：适用。

E. 产假
1. 资格与累积：所有正式全职女性员工可享受，每次生育可休105天，最多享受2次。
2. 申请：尽量在预产期前1个月提交申请，需直属主管批准。
3. 未使用假期：不可结转。
4. 折现：不可折现。
5. 试用期：适用。

F. 反暴力侵害妇女假
1. 资格与累积：遭受身体、性、心理伤害或威胁的女性员工可享受，每年10天。
2. 申请：尽快提交申请，需直属主管批准。
3. 未使用假期：不可结转。
4. 折现：不可折现。
5. 试用期：适用。

G. 单亲父母假
1. 资格与累积：单亲父母员工可享受，每年7天。
2. 申请：提前1天提交申请，需直属主管批准。
3. 未使用假期：不可结转。
4. 折现：不可折现。
5. 试用期：适用。

H. 丧假
1. 资格与累积：所有正式全职员工可享受丧假。员工在直系亲属（配偶、子女、父母、兄弟姐妹）去世时，每年可享受5天丧假。
2. 申请：应尽快通过员工自助门户提交休假申请，需直属主管批准。
3. 未使用假期：未使用的丧假不可结转至下一年，将自动作废。
4. 折现：未使用的丧假不可折现。
5. 试用期：试用期员工可享受丧假。

I. 陪审/公民义务假
1. 资格与累积：所有正式全职员工可享受陪审/公民义务假。休假时长与公民义务时长相同。
2. 申请：应尽快通过员工自助门户提交休假申请，建议至少在履行公民义务前1周提交，需直属主管批准。
3. 未使用假期：不适用（因属特定事件假期）。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

J. 学习/考试假
1. 资格与累积：所有正式全职员工每年可享受5天学习/考试假。
2. 申请：需提前1周通过员工自助门户提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工不享受此假期。

K. 慰问假
1. 资格与累积：当直系亲属（配偶、子女、父母、兄弟姐妹）患重病时，所有正式全职员工每年可享受3天慰问假。
2. 申请：应尽快通过员工自助门户提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

L. 学术休假
1. 资格与累积：连续服务满5年的正式全职员工，每5年可享受60天学术休假。
2. 申请：需提前3个月通过员工自助门户提交申请，需直属主管和人力资源部批准。
3. 未使用假期：不可结转至下一周期，将自动作废。
4. 折现：不可折现。
5. 试用期：试用期员工不享受此假期。

M. 婚假
1. 资格与累积：所有正式全职员工在职期间可享受一次5天婚假。
2. 申请：需提前1个月通过员工自助门户提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工可享受婚假。

N. 领养假
1. 资格与累积：所有正式全职员工在职期间可享受一次30天领养假。
2. 申请：应尽快通过员工自助门户提交申请，建议至少在预计领养日期前1个月提交，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

O. 兵役假
1. 资格与累积：所有正式全职员工可享受兵役假，休假时长与服役时长相同。
2. 申请：应尽快通过员工自助门户提交申请，建议至少在服役日期前1个月提交，需直属主管批准。
3. 未使用假期：不适用（因属特定事件假期）。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

P. 志愿服务假
1. 资格与累积：所有正式全职员工每年可享受2天志愿服务假。
2. 申请：需提前1周通过员工自助门户提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工不享受此假期。

Q. 家庭照顾假
1. 资格与累积：所有正式全职员工每年可享受7天家庭照顾假。
2. 申请：需提前1天通过员工自助门户提交申请，需直属主管批准。

R. 个人事假

S. 宗教节日假

T. 医疗预约假
1. 资格与累积：所有正式全职员工每年可享受2天医疗预约假。
2. 申请：需提前1天通过员工自助门户提交申请，需直属主管批准。
3. 未使用假期：不可结转，年度作废。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

U. 选举假
1. 资格与累积：所有正式全职员工可享受选举假，休假时长与选举工作所需时长相同。
2. 申请：应尽快通过员工自助门户提交申请，建议至少在选举日期前1周提交，需直属主管批准。
3. 未使用假期：不适用（因属特定事件假期）。
4. 折现：不可折现。
5. 试用期：试用期员工可享受此假期。

V. 无薪假
1. 资格与累积：所有正式全职员工在耗尽其他休假类型后，可申请无薪假，休假时长由直属主管和人力资源部决定。
2. 申请：需提前1周通过员工自助门户提交申请，需直属主管和人力资源部批准。
3. 未使用假期：不适用（因属特定事件假期）。
4. 折现：不适用。
5. 试用期：试用期员工可申请此假期。

IV. 公共假期
公共假期不计入任何休假类型。若员工休假期间遇到公共假期，其休假期限将自动延长一天。

V. 试用期休假
如前述规定，试用期员工可享受病假（按减半标准计算），但不享受年假和服务奖励假。

VI. 休假折现
如前述规定，年假和服务奖励假可在年底按基本工资标准折现。病假不可折现。

VII. 休假申请流程
1. 提交：所有休假申请必须通过员工自助门户提交。
2. 审批：所有休假申请需经直属主管批准。主管将综合考虑部门运营需求和员工休假余额后作出审批决定。
3. 提前通知：
   - 年假/服务奖励假：至少提前1天申请
   - 计划性病假：至少提前1天申请
   - 突发性病假：应尽快提交申请
4. 证明材料：连续病假超过2天需提供医疗证明。

VIII. 违规处理
违反本政策（包括未及时提交休假申请或提供虚假信息）将面临纪律处分，情节严重者可解除劳动合同。

IX. 政策修订
本政策每年进行审核，公司保留修改权利。

X. 疑问咨询
如有任何疑问，请联系人力资源部。

XI. 确认条款
继续雇佣关系即视为已阅读、理解并同意遵守本政策。

HR政策手册 - 考勤政策
I. 目的
本政策旨在明确全体员工考勤要求，规范公务外出和加班管理。

II. 适用范围
适用于公司所有正式全职员工，不分职位或部门。

III. 考勤政策
A. 公务外出
1. 资格与计算：所有正式全职员工可申请公务外出，时长与实际公务时间相同。
2. 申请：至少提前1天通过员工自助门户提交申请，需直属主管批准。
3. 未使用：不适用（按实际发生计算）。
4. 折现：不可折现。
5. 试用期：适用。

B. 加班
1. 资格与计算：所有正式全职员工可申请加班，时长按超出正常工作时间计算。
2. 申请：至少提前1天通过员工自助门户提交申请，需直属主管批准。
3. 未使用：不适用（按实际发生计算）。
4. 折现：可按基本工资加公司规定的补贴标准在月底折现。
5. 试用期：适用。

C. 培训发展假
1. 资格与计算：所有正式全职员工可申请，时长与培训项目时间相同。
2. 申请：至少提前1周通过员工自助门户提交申请，需直属主管批准。
3. 未使用：不适用（按实际发生计算）。

5. 试用期：适用。
